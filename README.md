# DU2Vox

**Deep Unfolding to High-Resolution Voxel Distribution Generation for Fluorescence Molecular Tomography**

## Overview

DU2Vox is a two-stage framework for FMT reconstruction:
- **Stage 1**: Coarse relative distribution initialization (MS-GDUN / GCAIN)
- **Stage 2**: FEM-consistent prior-guided voxel distribution generation (Residual INR) — *coming soon*

## Data

Training data is generated by [FMT-SimGen](https://github.com/foodszhang/FMT-SimGen).
Set `data.data_root` in your config to point to the FMT-SimGen output directory.

## Quick Start

```bash
# Train stage 1
python scripts/train_stage1.py --config configs/stage1/gcain_full.yaml

# Evaluate
python scripts/eval_stage1.py --config configs/stage1/gcain_full.yaml --checkpoint checkpoints/best.pth

# Visualize
python scripts/visualize.py --config configs/stage1/gcain_full.yaml --checkpoint checkpoints/best.pth --n_samples 4
```

## Training

Stage 1 training uses a 6-stage unrolled network with multi-scale graph attention.
Data is referenced via config `data_root` pointing to FMT-SimGen output directory.

## Evaluation

Per-foci and per-depth metric grouping available when `dataset_manifest.json` is present.
